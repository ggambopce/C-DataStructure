Queue 큐 
큐는 선입선출(FIFO: First In, First Out) 방식의 자료구조이며, 여기에서는 배열 기반으로 구현되었기 때문에 items[MAX]라는 고정 크기의 배열을 사용한다.
배열 큐는 메모리를 처음부터 고정된 크기로 확보하고, 삭제 시 값을 없애지 않고 단순히 front 포인터만 이동시켜서 논리적으로 값을 제거하는 것처럼 작동한다.

이 배열은 지역 변수일 경우 → 스택(Stack) 영역에 할당
전역 변수나 static일 경우 → 데이터(Data) 영역에 할당됨
크기는 MAX로 고정되며, 런타임 중 크기 변경 불가

삭제해도 items[front] 값을 0으로 덮거나 지우지 않음
그냥 front 값을 1 증가시켜 다음 값만 바라보게 함
→ 큐는 "값을 지우는" 게 아니라 "그 위치를 무시하는" 방식

| 메모리 영역    | 설명                              | 큐에 적용 시                             |
| --------- | ------------------------------- | ----------------------------------- |
| 스택(Stack) | 지역 변수, 함수 호출 시 자동 할당            | `main()` 안에서 선언된 `Queue q`          |
| 힙(Heap)   | 동적 메모리 할당 영역 (`malloc`, `free`) | `Queue *q = malloc(...)`로 동적 큐 만들 때 |
| 데이터 영역    | 전역/정적 변수 저장                     | `static Queue q;` 또는 전역 선언 시        |

포인터
메모리에 주소값을 저장할 수 있는 포인터 변수 공간(4바이트~8바이트)을 스택영역에 생성
동적 할당을 위한 힙 영역메모리에 참조할 수 있는 참조 변수 생성
포인터에 참조값을 &를 사용하여 저장 후 *을 사용하여 참조하여 변수사용
동적할당을 위한 함수, 구조체는 할당 해제가 필요
